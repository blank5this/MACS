# 月末/年末结账流程

## 月末结账

### 时间表
- T+1 至 T+3:业务部门提交单据(发货单、收料单、报销单)
- T+4 至 T+5:财务审核单据
- T+6:应收/应付账龄分析
- T+7:成本计算与分摊
- T+8:生成财务报表
- T+9:财务经理审核
- T+10:财务总监审批,关账完成

### 关键任务清单
- [ ] 库存盘点(A 类物料月度,B/C 类按计划)
- [ ] 银行对账单核对
- [ ] 应收账款账龄分析
- [ ] 应付账款账龄分析
- [ ] 折旧计算
- [ ] 工资分摊
- [ ] 水电费分摊
- [ ] 制造费用分摊
- [ ] 销售成本结转
- [ ] 损益结转

## 年末结账

### 额外任务
- [ ] 全面盘点 (12 月 31 日)
- [ ] 跌价准备评估
- [ ] 坏账准备评估
- [ ] 长期资产减值测试
- [ ] 所得税汇算清缴准备
- [ ] 关联方交易核对
- [ ] 财务会计报表审计 (外部审计师)

### 关账检查清单
- 所有会计分录已过账
- 试算平衡
- 银行余额调节表已编制
- 财务会计报表签字确认
- 备份关账数据

## 紧急联系人
- 系统故障:IT 运维
- 业务单据问题:业务部门主管
- 会计政策争议:财务总监
