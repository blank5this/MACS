# 应计与预提费用

## 概述
应计费用 (Accruals) 和预提费用 (Provisions) 遵循权责发生制原则,在费用归属期间确认,而不论实际收付时间。

## 应计费用
- **定义**:已发生但尚未收到发票/未支付的费用
- **常见场景**:
  - 已收货但未收到供应商发票
  - 已享受服务但未收到账单 (如水电费、电信费)
  - 员工已发生的差旅费,尚未报销

### 会计分录
```
借:管理费用/销售费用
   贷:应付账款/其他应付款
```

## 预提费用
- **定义**:预先计提但尚未发生的费用
- **常见场景**:
  - 设备大修费用(每年预提,实际发生时冲销)
  - 销售返利(根据销售业绩预提)
  - 利息费用(跨期分摊)

### 会计分录
```
借:管理费用/财务费用
   贷:预提费用
```

## 月末处理流程
1. 财务部每月最后 2 个工作日启动关账流程
2. 业务部门提交本月应计/预提清单
3. 财务审核 → 生成会计分录
4. 月末过账 → 编制财务报表

## 注意事项
- **谨慎性原则**:预提金额不得大幅超过实际可能发生额
- **追溯调整**:发现重大差错时,需调整以前年度损益
- **审计要求**:大额预提需有充分依据文件支持
